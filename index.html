Cómo funciona

  1. Elige el proveedor de IA y pega tu API key (queda solo en este navegador, nunca se envía a otro lado que no sea la API elegida).
  2. Sube el manual original (.docx, .pdf con texto seleccionable, .txt) o pega el texto directamente.
  3. Presiona «Traducir y maquetar». El modelo traduce al español de Chile con el mismo tono y estructura de tus manuales FDV de referencia.
  4. Revisa el borrador editable, sube las imágenes/diagramas donde corresponda, y presiona «Generar vista de impresión» para exportar a PDF.

Limitaciones: los PDF escaneados (imágenes de texto chino) no se pueden extraer automáticamente — necesitan OCR aparte. Los .doc antiguos no son compatibles, conviértelos a .docx o PDF antes de subir.
